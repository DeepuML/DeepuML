## 🚀 Major Projects

### 🎬 YouTube Comments Analysis — NLP at Scale
> *End-to-end NLP pipeline for large-scale YouTube comment intelligence*

| Layer | Stack |
|-------|-------|
| **Ingestion** | YouTube Data API v3 · Apache Kafka · Python |
| **Processing** | HuggingFace Transformers · VADER · spaCy · Pandas |
| **ML Models** | BERT fine-tuned sentiment · Topic modelling (LDA + BERTopic) · Toxicity detection |
| **Serving** | FastAPI · Redis cache · Docker |
| **Observability** | MLflow experiment tracking · Prometheus + Grafana dashboards |
| **Infra** | Kubernetes · GitHub Actions CI/CD |

**Highlights:**
- 🔍 Multi-label sentiment classification (positive / negative / neutral / toxic)
- 📊 Real-time topic extraction and trend surfacing across millions of comments
- ⚡ < 200 ms latency at p99 via async inference + Redis caching
- 📈 Drift detection with automated retraining triggers via Airflow DAGs

### 🏥 Health RAG — Production-Grade Retrieval-Augmented Generation
> *Medical knowledge retrieval system built for reliable, hallucination-resistant answers*

| Layer | Stack |
|-------|-------|
| **Data** | PubMed / medical corpus · Document chunking · Metadata tagging |
| **Embeddings** | BioBERT · sentence-transformers · FAISS / Chroma vector store |
| **Retrieval** | Hybrid BM25 + dense retrieval · Re-ranking (CrossEncoder) |
| **Generation** | LLaMA-3 / GPT-4 with constrained prompting · LangChain orchestration |
| **Serving** | FastAPI · Streaming responses · Auth middleware |
| **MLOps** | DVC data versioning · MLflow model registry · Docker Compose · K8s |

**Highlights:**
- 🏥 Domain-specific embeddings trained on biomedical literature for high-precision retrieval
- 🔒 Faithfulness guardrails — citation grounding and confidence scoring on every answer
- 🔄 Continuous document ingestion pipeline with delta updates
- 📦 Production hardened: rate limiting, logging, health checks, horizontal scaling
